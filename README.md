**Aura AI Studio** is a premium, high-fidelity neural network laboratory designed for interactive deep learning synthesis. Featuring a state-of-the-art "Aura Dark" design system, 3D holographic visualizations, and an integrated AI Architect, Aura provides an immersive environment for exploring the core mechanics of artificial intelligence.

---

## ✨ Core Ecosystem

### 🤖 1. Aura Assistant & Chat
The heart of the studio. An always-online AI Architect powered by **OpenAI** and **Gemini** to provide real-time technical synthesis, code assistance, and neural theory guidance.

### 🎮 2. Deep Playfield
An interactive playground for rapid neural prototyping. Experiment with dynamic parameters and watch high-fidelity signal flows in real-time.

### 🧠 3. Neural Engines
Master the foundations with precision:
- **Perceptron Engine**: Binary linear classification with live convergence dashboards.
- **Forward Synapse**: Non-linear signal flow through deep architectural layers.
- **Backward Gradient**: Error backpropagation and weights optimization.
- **Descent Optimization**: Visualizing loss landscapes with Gradient Descent.

### 🧬 4. Advanced Systems
- **Associative Memory**: Pattern recall within **Hopfield** energy basins.
- **Semantic NLP**: Sequence modeling and sentiment intelligence via **LSTM** networks.
- **Deep Belief Network**: Stacked RBM pretraining and latent feature extraction.

### 👁️ 5. Optical Intelligence (OpenCV Hub)
A powerhouse of computer vision modules with real-time detection pipelines:
- **Face Scanner & Attendance**: Biometric identification and logging.
- **Vehicle & Sign Detection**: Powered by high-performance **YOLOv8**.
- **Palm Reading**: Hand landmark tracking via **MediaPipe**.
- **Detection Gallery**: Comprehensive real-time visual analytics.

## 🎨 Design Philosophy: Aura Dark
Implemented with a bespoke CSS design system, Aura AI Studio offers:

## 🛠️ Quick Start

### 1. Initialize Workspace
```bash
git clone https://github.com/prateekpawar0226/Neural-Network-Tool-Box.git
cd Neural-Network-Tool-Box
```

### 2. Configure Environment
*Recommended: Python 3.11+*
```bash
pip install -r requirements.txt
```

### 3. Launch the Studio
```bash
streamlit run app.py
```
